## Usage

```shell
sudo apt-get install conky-all
git clone https://github.com/AbangSD/conky.git
cd linux-tool/conky
cp -r ./.conky ~
cp ./.conkyrc ~
cp ./conky.desktop ~/.config/autostart
conky
```
